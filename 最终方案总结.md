## 🎯 推荐方案（最简单）

### 使用 GitHub Actions 自动构建（无需 Mac！）

#### 步骤：

1. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 创建仓库，名称如 `antsword-build`

2. **上传代码到 GitHub**
   ```bash
   cd antsword-build
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/您的用户名/antsword-build.git
   git branch -M main
   git push -u origin main
   ```

3. **在 GitHub 上手动触发构建**
   - 进入仓库 → Actions
   - 选择 "Build AntSword"
   - 点击 "Run workflow" → 确认
   - 等待 10-15 分钟

4. **下载构建结果**
   - 在 Actions 页面找到完成的构建
   - 下载 `AntSword-macOS-arm64`
