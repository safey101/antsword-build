
# AntSword macOS ARM64 构建说明

## 环境准备
✅ Node.js 已安装 (v24.16.0)
✅ 项目依赖已配置完成

## 重要说明

**AntSword 官方推荐使用方式**：
- 使用 AntSword-Loader 加载器 + 源代码
- 官方只有 x64 版本的 Loader，没有 ARM64 版本

### 构建环境（独立配置）
位置：`c:\Users\admin\Documents\code\antsword-build`

## 构建步骤

### 方法 1：使用 npm 脚本（推荐在 macOS 上）
```bash
cd antsword-build
npm install
npm run build:mac:arm64
```

### 方法 2：使用 Node.js 脚本
```bash
cd antsword-build
npm install
node run-build.js
```

## 注意事项

### 在 Windows 上构建 macOS 应用
- electron-builder 支持跨平台构建，但：
  - 需要 macOS 代码签名证书
  - 部分 macOS 特定功能可能受限
  - 建议在真实的 macOS 环境（Apple Silicon）上构建

### 在 macOS 上构建
1. 安装 Xcode Command Line Tools
2. 安装 Node.js
3. 运行上述构建命令

## 官方替代方案

由于没有官方 macOS ARM64 的 AntSword-Loader，您可以：

1. **使用 Rosetta 2 转译**（在 Apple Silicon Mac 上）：
   - 下载官方 x64 版本的 AntSword-Loader
   - macOS 会自动使用 Rosetta 2 运行

2. **从源码编译 Loader**：
   - AntSword-Loader 的源码同样是 Electron 应用
   - 可以修改配置后打包为 ARM64 版本
