## 📋 接下来的步骤

### 第 1 步：在 GitHub 上创建仓库

1. 访问：https://github.com/new
2. 仓库名称：`antsword-build`
3. 选择 Public 或 Private（都可以）
4. **不要**勾选 "Initialize this repository with a README"
5. 点击 "Create repository"

### 第 2 步：推送代码到 GitHub

在 `antsword-build` 文件夹中打开 PowerShell 或命令提示符，执行：

```powershell
# 进入文件夹
cd c:\Users\admin\Documents\code\antsword-build

# 添加远程仓库（替换 safay101 为您的用户名）
git remote add origin https://github.com/safay101/antsword-build.git

# 重命名分支为 main
git branch -M main

# 推送到 GitHub
git push -u origin main
```

**注意**：推送时 GitHub 会要求您认证，按提示操作即可。

### 第 3 步：在 GitHub 上触发构建

推送成功后：

1. 访问您的仓库：https://github.com/safay101/antsword-build
2. 点击上方的 **Actions** 标签
3. 在左侧选择 **Build AntSword** 工作流
4. 点击右侧的 **Run workflow** 按钮
5. 点击绿色的 **Run workflow** 确认
6. 等待 10-15 分钟 ⏳

### 第 4 步：下载构建结果

构建完成后：

1. 进入 Actions → 选择刚完成的构建记录
2. 滚动到页面底部的 **Artifacts** 区域
3. 下载 **AntSword-macOS-arm64** 文件

---

## 📦 构建的内容

GitHub Actions 会同时构建：

- 下载的文件是 ZIP 格式，解压即可使用
