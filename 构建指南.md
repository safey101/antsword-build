
# AntSword macOS ARM64 构建完整指南

## 📋 前置准备

### 1. 系统要求
- **推荐**：Apple Silicon 芯片的 Mac (M1/M2/M3/M4)
- **可选**：Intel Mac (可以交叉编译 arm64)
- Node.js 14+

### 2. 在 macOS 上构建（推荐）

#### 步骤 1：安装 Node.js
```bash
# 使用 Homebrew
brew install node

# 或从官网下载
# https://nodejs.org/
```

#### 步骤 2：准备项目
```bash
# 将 antsword-build 文件夹复制到 Mac 上
cd antsword-build
```

#### 步骤 3：配置 npm 镜像
```bash
# 配置 Electron 镜像（国内加速）
npm config set electron_mirror https://npmmirror.com/mirrors/electron/
```

#### 步骤 4：安装依赖
```bash
npm install
```

#### 步骤 5：构建 macOS ARM64 版本
```bash
npm run build:mac:arm64
```

## 🎯 构建命令详解

| 命令 | 功能 |
|------|------|
| `npm run build:mac:arm64` | 构建 macOS ARM64 版本 |
| `npm run build:mac:x64` | 构建 macOS x64 版本 (Intel) |
| `npm run build:mac:universal` | 构建 macOS Universal 版本 (通用) |
| `npm run build:win` | 构建 Windows 版本 |
| `npm run build:linux` | 构建 Linux 版本 |

## 📦 构建输出

构建完成后，安装包位于 `dist/` 目录：
- `AntSword-2.1.16-arm64.dmg` - DMG 安装包
- `AntSword-2.1.16-arm64-mac.zip` - ZIP 压缩包

## ⚠️ 注意事项

### 在 Windows 上构建 macOS 应用
虽然 electron-builder 支持跨平台构建，但建议：
1. 在真实 Mac 上构建，避免兼容性问题
2. macOS 代码签名需要 Apple 开发者证书
3. 某些 macOS 特定功能只能在 macOS 上完成

### 代码签名
如果需要分发应用，需要配置 macOS 代码签名：
```json
// 在 package.json 中添加
"build": {
  "mac": {
    "identity": "Developer ID Application: Your Name (TEAM_ID)"
  }
}
```

## 🚀 快速开始（在 Mac 上）

```bash
# 1. 复制项目到 Mac
# 2. 进入目录
cd antsword-build

# 3. 配置镜像（可选，国内推荐）
export ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/

# 4. 安装依赖
npm install

# 5. 构建 ARM64 版本
npm run build:mac:arm64
```

## 🔧 故障排除

### 问题：下载 Electron 超时
```bash
# 使用镜像
export ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install
```

### 问题：构建失败
- 确保有足够的磁盘空间（至少 2GB）
- 删除 node_modules 和 package-lock.json，重新安装
- 检查 Xcode Command Line Tools 是否已安装
```bash
xcode-select --install
```

### 问题：找不到 node 命令
确保 Node.js 已正确安装：
```bash
node --version
npm --version
```
